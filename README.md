## üéµ How It Works

The application uses the Web Audio API to procedurally generate music based on carefully crafted presets. Each preset defines:

- **Musical Characteristics**: Tempo, instruments, atmosphere, and key elements
- **Mood**: Emotional qualities that shape the composition
- **Audio Parameters**: Synthesis settings that recreate the sonic signature of each source material

When you select a preset and click "Generate Music", the application:
1. Analyzes the preset's musical characteristics
2. Creates oscillators, filters, and effects chains
3. Synthesizes audio in real-time
4. Plays the generated composition through your speakers
